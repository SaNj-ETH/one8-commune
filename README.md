# One8 Commune - Indore

A clean, aesthetic, mobile-friendly restaurant ordering webapp for One8 Commune Indore.

## Features
- **Digital Menu**: Browse categories and items with a clean, premium design.
- **Cart System**: Add items, adjust quantities, and view total.
- **Checkout**: Simple checkout flow (simulated).
- **Mobile First**: Optimized for mobile devices with touch-friendly controls and bottom sheets.

## Tech Stack
- React (Vite)
- Vanilla CSS (Custom Design System)
- Lucide React (Icons)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```

## Note on Data
The menu data is located in `src/data/menu.json`. Due to the dynamic nature of the source website and access restrictions ("Restaurant Closed"), a representative sample of the menu has been created. You can easily expand this JSON file with the full menu data following the existing structure.
